Intro to importing generated docs from external projects

#How to import generated content from external repos?

Unlike static content generated content is not available right off from the external repo.

The suggested approach is as follows:

  1. On external project set up scripts/hooks that
    1. generate docs into markdown
    2. commit generated docs into an orphaned commit and associated dedicated branch
    3. push orphaned commit to remote
    4. emit publication event for centralized docs project
    5. intercept delete branch event and delete dedicated branch
  2. On centralized documentation project:
    1. connect external repo with subrepo clone ... or intercept publication event
    2. import remote dedicated branch with subrepo pull ... && subrepo clean ...
    3. emit delete branch event for external project
    4. build and publish centralized docs
